# TicTacToe

A simple Tic Tac Toe game implemented in Java and run through the terminal.

---

## Requirements

* Java (JDK 8 or higher)

Check installation:

```bash
java -version
javac -version
```

---

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/shubham-p-prateek/tictactoe.git
cd tictactoe/src
```

---

### 2. Compile the program

```bash
javac TicTacToe.java
```

---

### 3. Run the program

```bash
java TicTacToe
```

## Notes

* The program runs entirely in the terminal
* Make sure you are inside the `src` folder before compiling
* The class name must match the filename (`TicTacToe`)

---

## Common Errors

"Could not find or load main class"

* You are not in the correct directory
* You compiled or ran with the wrong class name

"javac: command not found"

* Java is not installed or not added to PATH
